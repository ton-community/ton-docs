# Toncoin: native token

The native cryptocurrency of TON Ecosystem is **Toncoin**. Transaction fees, gas fees, and storage fees are paid in Toncoin. Toncoin is used for staking to become a validator.

- The process of making Toncoin payments is described in [Payments processing](/v3/guidelines/dapps/asset-processing/payments-processing).
- You can find where to buy or exchange Toncoin on the [Toncoin page](https://ton.org/coin).
