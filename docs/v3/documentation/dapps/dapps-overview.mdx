# DApps Overview

TON decentralized applications (DApps) are powered by on-chain smart contracts and may integrate off-chain components for UI and data access. They support a range of use cases, including token transfers, payments, decentralized finance, and external data integration.

This page provides an overview of the core primitives used in DApp development on TON: digital assets, DeFi principles, and blockchain oracles.
