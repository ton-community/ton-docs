# Testnet

Testnet is an independent blockchain network that replicates Mainnet functionality using valueless tokens.

Use Testnet to safely develop and debug before deploying to Mainnet.

:::info
Toncoin and other assets on Testnet have no value, and the network can be reset at any time.
:::
