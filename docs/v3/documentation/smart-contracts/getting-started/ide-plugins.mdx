# IDE plugins

Select your IDE or editor and install the relevant plugin for your language.

There are officially supported plugins for [Tolk](/v3/documentation/smart-contracts/tolk/overview), [FunC](/v3/documentation/smart-contracts/func/overview), [Fift](/v3/documentation/smart-contracts/fift/overview), [TL-B](/v3/documentation/data-formats/tlb/overview), and [Blueprint](/v3/documentation/smart-contracts/getting-started/javascript).

## JetBrains IDEs

Plugin for JetBrains products (IntelliJ IDEA, WebStorm, PyCharm, CLion, etc.).

![TON JetBrains plugin screenshot](/img/docs/ton-jetbrains-plugin.png)

There are several ways to install a plugin:
- Using an IDE built-in plugin system:
<kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "TON"</kbd> >
  <kbd>Install Plugin</kbd>

- Via the [Marketplace](https://plugins.jetbrains.com/plugin/23382-ton)
- Download the [latest release](https://github.com/ton-blockchain/intellij-ton/releases/latest) and install it manually using
<kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

## Visual Studio Code

Extension for Visual Studio Code (VS Code) and VS Code-based editors (VS Codium, Cursor, Windsurf, etc.).

### Marketplace

Install the TON extension in [VS Code](https://marketplace.visualstudio.com/items?itemName=ton-core.vscode-ton) or [VS Code-based editors](https://open-vsx.org/extension/ton-core/vscode-ton).

### Manually
Get the latest `.vsix` file from [releases](https://github.com/ton-blockchain/ton-language-server/releases), from
[VS Code marketplace](https://marketplace.visualstudio.com/items?itemName=ton-core.vscode-ton),
or from [Open VSX Registry](https://open-vsx.org/extension/ton-core/vscode-ton).

In VS Code:
<kbd>Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)</kbd> >
<kbd>Type "Install from VSIX"</kbd> >
<kbd>Select the downloaded `.vsix` file</kbd> >
<kbd>Reload VS Code</kbd>

## Web IDE
Using the [TON Web IDE](https://ide.ton.org/), you can try smart contract development online without installing anything locally.
