# Toncoin bridge

The Toncoin bridge enables transfers of Toncoin between TON and Ethereum Blockchain and between TON and the BSC (BNB Smart Chain).

This bridge is managed by [decentralized oracles](/v3/documentation/infra/bridges/toncoin-addresses).

### How to use it

The bridge frontend is available at [ton.org/bridge](https://ton.org/bridge).

:::info
[Bridge frontend source code](https://github.com/ton-blockchain/bridge)
:::
