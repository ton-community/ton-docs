# Cross-chain bridges

Decentralized cross-chain bridges operate on the TON blockchain, enabling asset transfers between TON and other blockchains.

## Toncoin bridge

The Toncoin bridge enables Toncoin transfers between TON and Ethereum, and between TON and BNB Smart Chain (BSC).

This bridge is managed by a decentralized set of [oracles](/v3/documentation/infra/crosschain/bridge-addresses).

### How to use it

The bridge frontend is available at [ton.org/bridge](https://ton.org/bridge).

:::info
[Bridge frontend source code](https://github.com/ton-blockchain/bridge)
