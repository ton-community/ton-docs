# Introduction

**The Open Network (TON)** is a decentralized platform designed to power a user-owned Internet.
It brings together key technologies — _TON Blockchain, TON DNS, TON Storage, TON Sites_, and _TON Proxy_ — into one cohesive ecosystem.

**TON Blockchain** is the core of TON, built as a distributed supercomputer that powers services and applications for a decentralized Internet. It aims to support millions of users and applications worldwide.

Start exploring TON through key entry points.
Understand the blockchain, discover core apps, compare with Ethereum, or dive into learning resources.
