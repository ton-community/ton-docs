# Connect with TON Proxy

TON Proxy works with standard HTTP proxy settings so that you can configure it directly through your browser or operating system.

## Browsers

### Safari
Safari uses your operating system’s proxy settings. Follow: [macOS](#macos) or [iOS](#ios).

### Google Chrome
Google Chrome does not have its own proxy settings. Configure the proxy at the **operating system** level; Chrome will automatically use it. 

### Firefox
1. Go to **Settings → General → Network Settings → Configure**.  
2. Choose **Manual proxy settings**.  
3. In **HTTP Proxy**, enter the address of a public entry proxy.  
4. In **Port**, enter `8080` (without quotes).  
5. Click **OK**.

## Operating systems

### Windows
1. Open **Settings → Network & Internet → Proxy**.  
2. Under **Manual proxy setup**, select **Set up** and turn on **Use a proxy server**.  
3. Enter the public entry proxy in **Address** and `8080` in **Port**.  
4. Click **Save**.

### Ubuntu (Linux)
1. Open **Settings → Network**.  
2. Click **Network Proxy → Manual**.  
3. Enter the public entry proxy in **HTTP Proxy** and `8080` in **Port**.

### Android
1. Go to **Settings → Wi-Fi** and long-press your network.  
2. Select **Modify network → Advanced options → Manual**.  
3. Enter the public entry proxy in **Server** and `8080` in **Port**.  
4. Tap **Save**.

### macOS
1. Go to **Settings → Network → Advanced → Web proxy (HTTP)**.  
2. In **Web proxy server**, enter the public entry proxy, followed by `:8080`.  
3. Click **OK**.

### iOS
1. Go to **Settings → Wi-Fi** and tap your current network.  
2. Under **HTTP Proxy**, choose **Manual**.  
3. Enter the public entry proxy in **Server** and `8080` in **Port**.  
4. Tap **Save**.
