# Connect with TON Proxy

TON Proxy is fully compatible with standard HTTP proxy settings so that you can configure it directly through your browser or operating system.

## Google Chrome

Follow the instructions specific to your device’s operating system: Windows, macOS, Linux, iOS, or Android.

## Firefox

1. Go to **Settings → General → Network Settings → Settings...**. 
2. Choose **Manual proxy configuration**. 
3. In **HTTP Proxy**, enter the address of a public entry proxy. 
4. In **Port**, enter 8080.
5. Click **OK**.


## Safari

Safari is available on Apple platforms. Follow the instructions for **[macOS](#macos)** or **[iOS](#ios)**.

## iOS

1. Go to **Settings → Wi‑Fi**. 
2. Tap your currently connected network. 
3. Scroll to **HTTP Proxy** and choose **Manual**. 
4. In **Server**, enter the address of a public entry proxy. 
5. In **Port**, enter 8080. 
6. Tap **Save**.

## Android

1. Go to **Settings → Wi‑Fi**.
2. Tap and hold the Wi‑Fi network name. 
3. Select **Modify Network → Advanced options → Proxy → Manual**. 
4. In **Server**, enter the address of a public entry proxy. 
5. In **Port**, enter 8080.
6. Tap **Save**.

## Windows

1. Click **Start** and select **Settings → Network & Internet → Proxy**. 
2. Under **Manual proxy setup**, next to **Use a proxy server**, select **Set up**. 
3. Turn on **Use a proxy server**.
4. Enter the proxy address in **Address** and 8080 in **Port**.
5. Click **Save**.

## macOS

1. Go to **Settings → Network → Advanced → Web proxy (HTTP)**. 
2. In **Web proxy server**, enter the address of one of the public entry proxies; after the colon, enter 8080.
3. Click **OK**.

## Ubuntu

1. Go to **Settings → Network**. 
2. Click **Network Proxy → Manual**. 
3. In **HTTP Proxy**, enter the address of one of the public entry proxies.
4. In **Port**, enter 8080.
