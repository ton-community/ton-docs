# 토큰 (FT, NFT)

[TON 토큰 개요](https://telegra.ph/Scalable-DeFi-in-TON-03-30)

> TON의 토큰과 NFT는 중앙 집중식 실패 지점과 병목 현상을 피하고 분산화되어 있습니다.
>
> 컬렉션의 각 NFT는 별도의 스마트 컨트랙트입니다. 각 사용자의 토큰 잔액은 별도의 사용자 지갑에 저장됩니다.
>
> 스마트 컨트랙트는 서로 직접 상호 작용하여 전체 네트워크에 부하를 분산시킵니다.
>
> 사용자와 거래가 늘어나도 네트워크는 균일한 부하를 유지하여 원활한 확장성을 제공합니다.

### Jetton deployer

제톤은 TON 블록체인의 커스텀 대체 가능 토큰입니다. 다음 제톤 배포자 예시를 사용하여 TON 블록체인에서 자신만의 토큰을 만들 수 있습니다:

- **[TON Minter](https://minter.ton.org/)** — 오픈소스 제톤 배포자 dApp
- [제톤 배포자 — 컨트랙트](https://github.com/ton-defi-org/jetton-deployer-contracts) (FunC, TL-B)
- [제톤 배포자 — 웹클라이언트](https://github.com/ton-defi-org/jetton-deployer-webclient) (React, TypeScript)
