# Connect a wallet

지갑 개발자는 TON Connect에 지갑을 연결하여 사용자가 TON 앱과 안전하고 편리하게 상호작용할 수 있도록 할 수 있습니다.

## 통합

지갑을 TON Connect에 연결하려면 다음 단계를 따르세요:

1. [프로토콜 명세](/v3/guidelines/ton-connect/overview/)를 주의 깊게 읽으세요.
2. [SDK](/v3/guidelines/ton-connect/guidelines/developers) 중 하나를 사용하여 프로토콜을 구현하세요.
3. [wallets-list](https://github.com/ton-blockchain/wallets-list)에 풀 리퀘스트로 지갑을 추가하세요.
