# Blueprint

![Blueprint](/img/blueprint/logo.svg)

محیط توسعه برای TON برای نوشتن، تست و استقرار قراردادهای هوشمند.

## Quick start

دستورات زیر را در ترمینال اجرا کنید تا یک پروژه جدید ایجاد کرده و دستورات روی صفحه را دنبال کنید:

```bash
npm create ton@latest
```

### ویژگی‌های اساسی

- روند ساده‌سازی شده برای ساخت، تست و استقرار قراردادهای هوشمند
- استقرار بسیار ساده بر روی mainnet/testnet با استفاده از کیف پول مورد علاقه شما (به عنوان مثال، Tonkeeper)
- تست بسیار سریع قراردادهای هوشمند متعدد در یک بلاک چین ایزوله که در فرآیند در حال اجرا است

### پشته فناوری

- کامپایل FunC با استفاده از https://github.com/ton-community/func-js (بدون cli)
- Compiling Tact with https://github.com/tact-lang/tact
- تست قراردادهای هوشمند با استفاده از https://github.com/ton-org/sandbox
- Deploying smart contracts with TON Connect 2.0 compatible wallets or a `ton://` deeplink

### نیازمندی‌ها

- [Node.js](https://nodejs.org/) با نسخه‌ای جدید مثل v18، نسخه را با `node -v` بررسی کنید
- IDE with TypeScript and TON support:
  - [Visual Studio Code](https://code.visualstudio.com/) with the [FunC plugin](https://marketplace.visualstudio.com/items?itemName=tonwhales.func-vscode) or [Tact plugin](https://marketplace.visualstudio.com/items?itemName=tonstudio.vscode-tact)
  - [IntelliJ IDEA](https://www.jetbrains.com/idea/) with the [TON Development plugin](https://plugins.jetbrains.com/plugin/23382-ton)
