# О TON Connect

**TON Connect** — это протокол с открытым исходным кодом, который пользователям безопасно и удобно входить в TON-приложения, используя свои кошельки вместо традиционных паролей.

## Как работает TON Connect

TON Connect позволяет **кошелькам** и **DApps** (децентрализованным приложениям) в экосистеме взаимодействовать.

### Модель взаимодействия

- **Кошельки** предоставляют пользовательские интерфейсы для утверждения транзакций и надёжно хранят криптографические ключи на пользовательских устройствах.
- **DApps** в TON предлагают богатую функциональность и надёжно хранят пользовательские средства с помощью смарт-контрактов.

Это разделение ответственности способствует стремительным инновациям и надёжности: *кошельки не привязаны к закрытым экосистемам, а DApps не хранят пользовательские аккаунты*.

- **TON Connect** связывает кошельки и приложения, делая соединение бесшовным и надёжным для пользователей.

## Причины использовать TON Connect

- **Бесшовная интеграция**: позволяет подключить кошельки к DApps с минимальными сложностями.
- **Надёжная защита**: использует защищённую коммуникацию для защиты пользовательских данных и транзакций.
- **Удобство для пользователя**: не требуется дополнительных логинов, пользователи взаимодействуют с DApps прямо с помощью их кошельков.
- **Кроссплатформенность**: поддерживает веб, мобильные и десктопные приложения.
- **Инструменты для разработчиков**: предоставляет разработчикам набор инструментов и API для надёжной, консистентной интеграции.
- **Открытый исходный код**: разработка контролируется командой TonTech и открыта к участию со стороны.

### Для разработчиков DApp

TON Connect позволяет добавиться более тесной интеграции и лучшего отклика от пользователей:

- **Трафик**. Привлекайте больше пользователей с помощью криптокошельков, поддерживающих TON Connect.
- **Встроенная аутентификация**. Используйте кошельки пользователей как готовые учётные записи, не требуя регистрации.
- **Платежи**. Быстро и безопасно обрабатывайте платежи в Toncoin или USD₮ с помощью блокчейна TON.
- **Удержание**. Пользователям удобно снова посещать недавно открытые и избранные приложения.

### Для разработчиков кошельков

Подключите ваше приложение кошелька к TON Connect, чтобы обеспечить его надёжную и бесшовную интеграцию с DApps в TON.
Прочитайте, как [интегрировать TON Connect в ваш кошелек](/v3/guidelines/ton-connect/wallet/).

## Присоединяйтесь к экосистеме TON

Чтобы интегрировать свой сервис с экосистемой TON, вам необходимо сделать следующее:

- **Реализация TON Connect**: включите протокол TON Connect в своё приложение.
- **Совершить перевод**: создавайте и отправляйте сообщения с помощью библиотек TON. Детали реализации можете увидеть на странице [Отправка сообщений](/v3/guidelines/ton-connect/cookbook/ton-transfer).
- **Подключение платежей**: обрабатывайте платежи через ([TON API](https://tonapi.io/)) или собственный индексатор, например, [gobycicle](http://github.com/gobicycle/bicycle). Узнайте больше из нашего [руководства по обработке платежей](/v3/guidelines/dapps/asset-processing/payments-processing).
