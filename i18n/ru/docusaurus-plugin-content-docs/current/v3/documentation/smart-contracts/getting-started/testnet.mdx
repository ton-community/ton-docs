# Тестнет TON

Тестнет (testnet) в TON — это независимая блокчейн-сеть, которая копирует функциональность мейннета (основной сети), но использует токены без ценности.

Используйте тестнет для безопасной разработки и отладки перед развёртыванием в мейннете.

:::info
Монеты в тестовой сети не имеют ценности, а сама она может быть сброшена в любой момент.
:::
