# The Open Network

**The Open Network (TON)** — это децентрализованная и открытая платформа для создания интернета, принадлежащего пользователям.
Она соединяет несколько ключевых технологий — *TON Blockchain, TON DNS, TON Storage, TON Sites* и *TON Proxy* — в одну цельную экосистему.

**Блокчейн TON** — это основа TON. Он создан как распределённый суперкомпьютер для сервисов и приложений децентрализованного интернета. Он нацелен на поддержку миллионов пользователей и приложений по всему миру.

Начните исследовать TON с помощью одной из точек входа.
Разберитесь в блокчейне, исследуйте главные приложения, узнайте отличия от Ethereum или погрузитесь в образовательные ресурсы.
