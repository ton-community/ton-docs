# Эксплореры в TON

В этой статье мы рассмотрим эксплореры в TON и их возможности с точки зрения пользователя.

## Что такое эксплорер (обозреватель)?

Эксплорер, он же «обозреватель» — это веб-сайт, позволяющий просматривать информацию в блокчейне, такую как баланс счёта, историю транзакций, блоки.

## Какие эксплореры существуют?

Среди эксплореров в TON можно выделить несколько категорий:

- Для повседневного использования
- С расширенной информацией для разработчиков
- Специализированные

Это разделение на категории во многом условно, и один эксплорер может относиться к нескольким категориям одновременно. Поэтому не будем уделять этому слишком много внимания.

## Общая функциональность

Эксплореры позволяют видеть информацию из блокчейна и искать нужную, подобно тому, как в банковском приложении можно увидеть историю транзакций и баланс счёта. Они предоставляют удобный пользовательский интерфейс для доступа в реальном времени к данным о транзакциях, блоках и адресах.

Как с помощью банковского приложения можно отслеживать, на что вы потратили деньги, кто отправлял их вам и когда происходили транзакции, так и эксплореры позволяют видеть это для цифровых активов в блокчейне. Они важны для прозрачности, позволяя кому угодно независимо верифицировать транзакции, видеть криптовалютный поток и анализировать активность в блокчейне.

Как правило, эксплорер позволяет найти транзакцию или аккаунт по идентификатору. Некоторые также поддерживают DNS-адреса или собственную [адресную книгу](/v3/concepts/dive-into-ton/ton-ecosystem/explorers-in-ton#address-alias-in-explorers), что позволяет добавлять аккаунтам человекочитаемые метки. Пример: [TON Foundation](https://tonviewer.com/UQAFmjUoZUqKFEBGYFEMbv-m61sFStgAfUR8J6hJDwUU04VW).

## Эксплореры TON

### Tonscan

Хороший эксплорер для повседневного использования. Он обеспечивает комплексное представление блокчейна TON, позволяя пользователям искать транзакции, адреса, блоки и многое другое.

- Версия для мейннета: [tonscan.org](https://tonscan.org/)
- Версия для тестнета: [testnet.tonscan.org](https://testnet.tonscan.org/)

#### Возможности

- Удобен для повседневного использования и при разработке
- Продвинутые страницы для Jettons и пулов номинаторов
- Типы контрактов и дизассемблер
- Поиск по удобным адресам TON DNS и [адресной книге](https://github.com/catchain/address-book).

### Tonviewer

Хороший эксплорер для повседневного использования со сбалансированным UI для пользователей и разработчиков.

- Версия для мейннета: [tonviewer.com](https://tonviewer.com/)
- Версия для тестнета: [testnet.tonviewer.com](https://testnet.tonviewer.com/)

#### Возможности

- Удобен для повседневного использования и при разработке
- Подробная информация с трейсами для разработчиков
- Типы контрактов и дизассемблер
- Поиск по удобным DNS-адресам и меткам [tonkeeper/ton-assets](https://github.com/tonkeeper/ton-assets).

### Tonscan от Bastion

- Для мейннета: [tonscan.com](https://tonscan.com/)

#### Возможности

- Агрегированная аналитика транзакций
- Отслеживание транзакций

### DTON

DTON — ещё один эксплорер для разработчиков. Он предоставляет много информации о транзакциях в удобной форме.

Кроме того, он позволяет вам просматривать фазу вычислений транзакции шаг за шагом.

- Для мейннета: [dton.io](https://dton.io/)

#### Возможности

- Удобен для разработчиков
- Расширенная информация о фазе вычислений
- Типы контрактов и дизассемблер

### Эксплорер от Ton Whales

Эксплорер для разработчиков.

- Версия для мейннета: [tonwhales.com/explorer](https://tonwhales.com/explorer)

#### Возможности

- Удобен для разработчиков
- Типы контрактов и дизассемблер
- Простой и удобный просмотр деталей о валидаторах

### TON NFT Explorer

Этот эксплорер специализируется на NFT, однако его можно использовать и как универсальный.

- Версия для мейннета: [explorer.tonnft.tools](https://explorer.tonnft.tools/)
- Версия для тестнета: [testnet.explorer.tonnft.tools](https://testnet.explorer.tonnft.tools/)

При просмотре адреса кошелька можно узнать, какие NFT в нём хранятся, а при просмотре конкретной NFT можно увидеть метаданные, адрес коллекции, владельца, а также историю транзакций.

#### Возможности

- Удобен для разработчиков
- Специализируется на NFT
- Типы контрактов

## Кроссчейн-эксплореры

### OKX TON explorer

Этот эксплорер спроектирован, чтобы предоставлять подробные и детализированные данные из TON. Вне зависимости от того, отслеживаете вы адреса кошельков, исследуете Jettons, или погружаетесь в NFT-коллекции, он предлагает дружелюбный интерфейс, подходящий как начинающим, так и разработчикам.

- Мейннет: [web3.okx.com/explorer/ton](https://web3.okx.com/explorer/ton)

#### Возможности

- Удобен для повседневного использования
- Поддержка TON DNS
- Данные о Jetton и NFT

### TON NFTscan

Этот эксплорер разработан специально для невзаимозаменяемых токенов (NFT). Он позволяет исследовать, отслеживать и проверять транзакции, контракты и метаданные NFT.

- Для мейннета: [ton.nftscan.com](https://ton.nftscan.com/)

#### Возможности

- Удобен для активных пользователей
- Рейтинг коллекций NFT
- Полезная информация для трейдеров, например, ежедневные объёмы торгов

## Аналитические эксплореры

### TonStat

- Для мейннета: [tonstat.com](https://www.tonstat.com/)

Отображает различные статистические данные, такие как количество зарегистрированных сетевых адресов и кошельков, объем сжигаемых Toncoin, объем стейкинга Toncoin, объем выпущенных NFT, количество валидаторов и ряд других показателей.

#### Возможности

- Подходит для обычных пользователей
- Информация о DeFi
- Полезная информация для трейдеров, например, ежедневные объёмы торгов

## Псевдоним адреса в эксплорерах

Сделайте адрес своего сервиса удобнее для пользователей, используя алиас адреса. Для этого можно создать пулл-реквесты (PR) в соответствии с предоставленными ниже руководствами:

- [tonkeeper/ton-assets](https://github.com/tonkeeper/ton-assets) — псевдонимы Tonviewer.com
- [catchain/address-book](https://github.com/catchain/address-book) — псевдонимы Tonscan.org
- [ton-labels](https://github.com/ton-studio/ton-labels) — публичная база адресов в TON с лейблами
