# Блокчейн-сервисы

## Системы доменных имён

В Ethereum используют cервис доменных имён **Ethereum Name Service (ENS)** — децентрализованную систему на основе Ethereum.

А TON включает в себя систему доменных имён TON DNS. Cервис позволяет пользователям регистрировать понятные для человека доменные имена для смарт-контрактов, веб-сайтов или другого онлайн-контента. Это облегчает взаимодействие с **децентрализованными приложениями (DApps)** и другими ресурсами в блокчейне TON.

Система DNS в TON действует похоже на традиционные системы DNS в Интернете, но её децентрализованная природа устраняет необходимость в централизованном органе для контроля и управления доменными именами. Это снижает риски цензуры, мошенничества и перехвата доменов.

Одной из ключевых особенностей TON DNS является возможность прямой привязки криптокошельков к доменным именам. Это позволяет пользователям отправлять криптовалюту на адреса типа alice.place.ton без передачи дополнительных данных. Это упрощает пожертвования и платежи, делая процесс интуитивнее и удобнее.

## Прокси

TON Proxy — это инструмент на основе протокола TON, который обеспечивает высокий уровень безопасности и анонимности. Все данные, которые передаются через TON Proxy, зашифрованы, что защищает конфиденциальную информацию пользователей.

Одно из главных преимуществ TON Proxy — способность обходить блокировки интернет-провайдеров или государственных органов. Это ценный инструмент для пользователей, которым необходим свободный доступ к информации в интернете без ограничений.

Более того, TON Proxy также позволяет ускорить скорость подключения к интернету. Он автоматически выбирает серверы с наименьшей нагрузкой, что улучшает качество соединения.

## Децентрализованное хранилище

Ethereum не подходит для хранения больших объёмов данных. Поэтому под децентрализованным хранилищем на Ethereum обычно подразумеваются распределённые файловые системы, где хранить и извлекать данные можно децентрализованно и безопасно. Один из популярных подходов к децентрализованному хранению на Ethereum — InterPlanetary File System (IPFS): распределённая одноранговая файловая система.

В свою очередь, сеть TON имеет собственную децентрализованную службу хранения snapshots, которая используется блокчейном TON для хранения архивных копий блоков и данных о состоянии. Также она может быть использована для хранения пользовательских файлов или других служб на платформе TON, с доступом, подобным торрент-технологии. Самый популярный вариант использования такой системы — хранение метаданных NFT непосредственно в хранилище TON, без использования дополнительных распределённых хранилищ вроде IPFS.

## Платёжные сервисы

TON Payments — это решение для мгновенных транзакций с нулевыми сетевыми комиссиями на блокчейне TON. Для большинства задач достаточно самого блокчейна TON, однако некоторые приложения, такие как TON Proxy, TON Storage или определённые виды dApps, требуют микротранзакций с гораздо более высокой скоростью и меньшими затратами. Технология Payment channels может решать эту задачу.

Платёжные каналы позволяют двум сторонам совершать транзакции вне блокчейна, создавая специальный смарт-контракт на блокчейне со своими начальными балансами. Затем они могут выполнять неограниченное количество транзакций без каких-либо лимитов по скорости или комиссий. Сетевые сборы взимаются только при открытии и закрытии канала. Данная технология также гарантирует корректную работу между пользователями, позволяя одной стороне самостоятельно закрывать канал, если другая сторона обманывает или исчезает.
