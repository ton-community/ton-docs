Fift - это основанный на стеке язык программирования общего назначения, оптимизированный для создания, отладки и управления смарт-контрактами TON Blockchain.
Fift был специально разработан для взаимодействия с виртуальной машиной TON (TON VM или TVM) и блокчейном TON.

```fift
{ ."hello " } execute ."world"
hello world ok
```

:::info
Обычно использование языка Fift не требуется для программирования смарт-контрактов в TON. Однако иногда Вам может понадобиться использовать язык Fift для решения нестандартных технических задач в рамках Вашего задания.
:::
