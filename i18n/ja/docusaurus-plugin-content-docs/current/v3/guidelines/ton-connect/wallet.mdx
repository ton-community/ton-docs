# Connect a wallet

ウォレット開発者の場合は、ウォレットを TON Connect に接続し、ユーザーが安全かつ便利に TON アプリとやり取りできるようにすることができます。

## 統合

ウォレットをTON Connectに接続するには、以下の手順に従ってください。

1. format@@0(/v3/guidelines/ton-connect/overview/) を注意深く読んでください。
2. [SDKs](/v3/guidelines/ton-connect/guidelines/developers) のいずれかを使用してプロトコルを実装します。
3. プルリクエストでウォレットを [wallets-list](https://github.com/ton-blockchain/wallets-list)に追加します。
