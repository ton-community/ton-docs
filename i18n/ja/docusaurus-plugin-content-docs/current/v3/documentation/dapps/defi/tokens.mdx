# トークン（FT、NFT）

[分散型TONトークンの概要](https://telegra.ph/Scalable-DeFi-in-TON-03-30)

> TONを動力とするトークンとNFTは分散化されており、単一障害点とボトルネックを回避しています。
>
> コレクション内の各NFTは個別のスマートコントラクトです。各ユーザーのトークン残高は個別のユーザーウォレットに保管されます。 The token balance for each user is stored in a separate user wallet.
>
> スマートコントラクトは互いに直接やりとりし、ネットワーク全体に負荷を分散します。
>
> ユーザー数やトランザクション数が増加しても、ネットワークは均一な負荷を維持し、シームレスなスケーラビリティを実現します。

### Jetton deployer

Jettons are custom fungible tokens on TON Blockchain. JettonsはTON ブロックチェーン上のカスタムファンジブルトークンです。以下のJetton デプロイヤーの例を使って、TON ブロックチェーン上に独自のトークンを作成することができます：

- **[TON Minter](https://minter.ton.org/)** — オープンソースの Jetton デプロイヤー dApp
- [Jetton デプロイヤー — contracts](https://github.com/ton-defi-org/jetton-deployer-contracts) (FunC, TL-B)
- [Jetton デプロイヤー - WebClient](https://github.com/ton-defi-org/jetton-deployer-webclient) (React, TypeScript)

### Tools to work with jettons

- [NFT Jetton セールコントラクト](https://github.com/dvlkv/nft-jetton-sale-smc) - Jettonサポート付きNFTセールコントラクト
- [Scaleton](http://scaleton.io) — カスタムトークン残高を参照
- [@tegro/ton3-client](https://github.com/TegroTON/ton3-client#jettons-example)-ジェトンの情報を照会するSDK
