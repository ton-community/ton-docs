[Hacktoberfest](https://hacktoberfest.digitalocean.com/)は、_オープンソースプロジェクト_、その_メンテナー_、そして_コントリビューター_のコミュニティ全体を1ヶ月にわたって祝うものです。毎年10月、オープンソースのメンテナーは、開発者を最初のプルリクエストに導くため、新しいコントリビューターに細心の注意を払います。 Each October, open source maintainers give new contributors extra attention as they guide developers through their first pull requests.

TONコミュニティにとって、今こそエコシステムの成長を共に支援する時です。全世界を巻き込んで、**Hack-TON-berfest**パーティーを開催し、**年間ナンバーワンのオープンソース・エコシステム**を目指しましょう！

## 参加方法

2022 年の Hacktoberfest ルールは以下の通りです：

- **Hacktoberfest は誰でも参加できる！**
- 9月26日から10月31日の間、いつでも登録可能
- プルリクエストは、どのGITHUBまたはGITLABプロジェクトでも申請ができる：
  - [TONエコシステムのプロジェクト一覧](/hacktonberfest)
  - [GitHubのプロジェクト一覧](https://github.com/topics/hacktoberfest)
- 10月1日から10月31日までに、**４**件の pull/merge リクエストを受理する。
- Hacktoberfestを完了した最初の40,000人の参加者（メンテナーとコントリビューター）は、2つの賞品のどちらかを選ぶことができます：自分の名誉のために植えられた木、またはHacktoberfest 2022のTシャツ。(Hacktoberfestコミュニティより) _(from the Hacktoberfest community)_
- TONエコシステムプロジェクトの参加者（メンテナーおよびコントリビューター）全員に、[**限定Hack-TON-berfest NFT**](#what-the-rewards) が贈られます。(TON財団より) _(from TON Foundation)_

TONに関わるすべての人にとって、エコシステム全体の成長を促進し、TON財団から素晴らしい報酬を受け取るチャンスです。一緒に参加しましょう！ Let's do it together!

## 報酬は？

TONエコシステムのオープンソースプロジェクトへの貢献を促すために、TON財団から特別な報酬を受け取ることができます。参加者全員に、**限定Hack-TON-berfest NFT**を参加の証として差し上げます： Every participant will receive a **Limited Hack-TON-berfest NFT** achievement as a proof of participating:
