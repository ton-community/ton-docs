# TON 支付

TON 支付是微支付通道的平台。

它允许即时支付，无需将所有交易提交到区块链、支付相关交易费用（例如，为消耗的燃料支付费用）以及等待五秒钟，直到确认包含有关交易的区块。

因为此类即时支付的总费用如此之低，它们可以用于游戏、API 和链下应用程序中的微支付。[查看示例](/develop/dapps/defi/ton-payments#examples)。

- [TON 上的支付](https://blog.ton.org/ton-payments)
