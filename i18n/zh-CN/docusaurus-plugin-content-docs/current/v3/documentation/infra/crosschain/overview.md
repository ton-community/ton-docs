# 跨链桥

去中心化的跨链桥在TON区块链上运行，允许您将资产从TON区块链转移到其他区块链，反之亦然。

## Toncoin 跨链桥

Toncoin 跨链桥允许您在TON区块链和以太坊区块链之间，以及TON区块链和BNB智能链之间转移Toncoin。

跨链桥由[去中心化预言机](/participate/crosschain/bridge-addresses)管理。

### 如何使用？

跨链桥的前端托管在 https://ton.org/bridge。
